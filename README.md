# Minecraft: Beta Decorations Mod

A "vanilla plus" building expansion for Minecraft Beta 1.7.3 using Babric and StationAPI.

## Features
- Stone and wood variants.
- Stairs, slabs, and fences.
- Double slabs.

## Requirements
- The Babric mod loader.
- The StationAPI library mod.

## Philosophy
This mod overall is intended to feel indistinguishable from vanilla beta content.

## Notice
This mod is still in development and still remains unfinished. If you want to contribute, you are free to do so.
